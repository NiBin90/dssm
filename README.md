# dssm
## 数据分析
该数据集及其不平衡, simtrain_to05sts.txt中各个标签的分布如下：
{'0.0': 10098, '1.0': 1, '2.0': 11, '3.0': 21, '4.0': 90, '5.0': 2526, }
直接训练多分类模型，准确率低。因此，可将该问题转换为二分类问题，即标签值>2,标记为1;否则为0。
